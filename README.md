# Store API Workshop (Node.js + Express)

โปรเจกต์ Workshop การสร้าง RESTful API สำหรับระบบร้านค้า (E-commerce) เบื้องต้น 
รองรับระบบสมาชิก การจัดการสินค้า และการสั่งซื้อ

## Tech Stack
* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB (Mongoose)
* **Security:** bcrypt (Password Hashing), jsonwebtoken (JWT Authentication)

## การติดตั้งและรันโปรเจกต์

1. Clone project
   ```bash
   git clone https://github.com/ChuthamatDev/workshop-nodejs.git
   ```

2. Install independency
    ```bash
    npm i
    ```
3. Run project
    ```bash
    npm start
    ```
